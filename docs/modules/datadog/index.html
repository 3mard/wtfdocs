    Datadog

    Connects to the Datadog API and displays alerting modules.

    Configuration

    datadog:
  apiKey: "<yourapikey>"
  applicationKey: "<yourapplicationkey>"
  enabled: true
  monitors:
    tags:
      - "team:ops"
  position:
    top: 4
    left: 3
    height: 1
    width: 2
    Name Description Value
    apiKey Your DataDog API token. Your API key or leave it empty to use the WTF_DATADOG_API_KEY environment variable.
    applicationKey Your DataDog application key
    border Optional Whether or not to draw this widget with a border. Default: true. true, false
    enabled Optional Whether or not this module is executed and if its data displayed onscreen. Default: false. true, false
    monitors Configuration for the monitors functionality.
    position Defines where in the grid this module’s widget will be displayed.
    refreshInterval Optional How often, in seconds, this module will update its data. Default: 300. Any positive integer.
    tags List of tags you want to query monitors by.
